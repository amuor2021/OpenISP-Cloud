# OpenISP Cloud

OpenISP Cloud is an open-source cloud platform designed to help Internet Service Providers (ISPs) manage MikroTik networks from a modern web dashboard.

## Vision

Our mission is to provide affordable, open-source ISP management software for developing countries, starting with South Sudan and East Africa.

## Planned Features

- MikroTik Router Management
- Customer Management
- Hotspot Management
- Billing System
- Network Monitoring
- Multi-ISP Support
- REST API
- Docker Deployment
- AI-powered Network Assistant
